\doxysubsection{Detailed Description}
Represents a train composed of three wagons. 

Each wagon (\doxylink{class_wagon}{Wagon} object) has its own number of seats, ticket price, and fill percentage. This class allows calculating actual revenue and finding the wagon with the minimum expected revenue.
