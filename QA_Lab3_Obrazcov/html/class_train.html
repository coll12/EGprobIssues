Detailed Description

Represents a train composed of three wagons.

Each wagon (Wagon object) has its own number of seats, ticket price, and fill percentage. This class allows calculating actual revenue and finding the wagon with the minimum expected revenue.

◆ actualRevenue()

double Train::actualRevenue ( )

Calculates the actual revenue taking into account fill percentages.

Returns
Sum of wagon revenues multiplied by their fill percentages.

Calculation formula:

\[\text{actualRevenue} = \sum_{i=1}^{3} \left( \text{wagon}_i.\text{expectedRevenue}() \times \frac{\text{fillPercent}_i}{100} \right) \]
